üìé Learning Intentions & Success Criteria

Learning Intentions

Students will:

  • Create enemy sprites that move automatically on screen
  • Implement basic collision detection between game objects
  • Design and code game-over conditions
  • Understand the relationship between movement patterns and gameplay

Success Criteria

Students can:

  • Create an enemy sprite that moves automatically
  • Detect collisions between sprites using pygame.sprite.collide_rect()
  • Create a game-over condition that stops the game appropriately
  • Explain how collision detection works in their own words
